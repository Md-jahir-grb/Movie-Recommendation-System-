# 🎬 Ultimate Movie Discovery Engine

A premium, production-ready Hybrid Movie Recommendation Dashboard built using Python, Streamlit, and Scikit-Learn. The application utilizes Machine Learning algorithms alongside live API state synchronization to fetch trailers, metadata, and dynamic graphical cards.

## 🚀 Engineered Features
- **Glassmorphic UI Elements:** Custom premium styling layers built directly inside Streamlit using micro-CSS injection.
- **Content-Based Filtering Subsystem:** Powered by Scikit-Learn's `CountVectorizer` and token-distance vector matrices evaluated using **Cosine Similarity**.
- **Dynamic Live TMDB Sync:** Live asynchronous communication layers that fetch real-time analytics, dynamic trailers, and trending movie graphics directly via TMDB REST API.
